Kesamaan Ordered dan Unordered List

Kesamaan Ordered dan Unordered list adalah keduanya menggunakan li pada anggota list

UnOrderedList

  • AAA
    Menggunakan Strong
  • BBB
    Menggunakan Italic
  • CCC
    Menggunakan Del untuk delete
  • DDD
    Menggunakan u untuk Underline

OrderedList

1. Original

  1. AAA
    Menggunakan Strong
  2. BBB
    Menggunakan Italic
  3. CCC
    Menggunakan Del untuk delete
  4. DDD
    Menggunakan u untuk Underline

2. Start with Number 10

maka bentuknya ol start = 10

  1. AAA
    Menggunakan Strong
  2. BBB
    Menggunakan Italic
  3. CCC
    Menggunakan Del untuk delete
  4. DDD
    Menggunakan u untuk Underline

3. Reversed

maka bentuknya ol Reversed

  1. AAA
    Menggunakan Strong
  2. BBB
    Menggunakan Italic
  3. CCC
    Menggunakan Del untuk delete
  4. DDD
    Menggunakan u untuk Underline

4. Start with 10 Reversed

maka bentuknya ol start = 10 Reversed

  4. DDD
