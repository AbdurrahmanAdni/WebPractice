Catatan

1. Class bisa digabung.

2. Class berbeda dengan id.

Class bisa digunakan oleh beberapa benda. Sehingga ada beberapa benda (dengan class yang sama) akan memiliki Style yang sama.

sedangkan pada id, Tidak boleh ada benda dengan id yang sama. Sehingga satu style pada suatu id hanya bisa digunakan oleh id tersebut saja.
